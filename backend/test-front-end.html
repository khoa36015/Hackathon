Auth API Tester

Dùng để test nhanh các endpoint /api/register, /api/login, /api/logout và kiểm tra session.

Trạng thái: —
Gợi ý CORS
app.after_request(add_cors_headers) đang bật; nếu dùng cookie cross-origin, cần Access-Control-Allow-Credentials: true và credentials: 'include'.
Login GET
GET /api/login trả gợi ý; dùng POST với JSON {username,password} để đăng nhập.
Lưu ý: nếu test qua file file:// có thể bị chặn CORS. Hãy mở bằng một server tĩnh đơn giản, ví dụ python -m http.server 5500 rồi vào http://127.0.0.1:5500/frontend.html.
